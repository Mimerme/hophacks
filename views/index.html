This project is intended to clearly provide information to those who need it.

For those in need of shelter, we hope this tool can connect people with what they need as quickly as possible. For those running shelters, we hope this tool provides a faster way to assess where resources are needed most. For law enforcment agencies hoping to get people into appropriate housing, this tool quickly provides a functional guide.
